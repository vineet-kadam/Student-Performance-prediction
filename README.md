# Student-Performance-prediction
1. Overview
This project predicts whether a student will pass or fail using a Logistic Regression model based on academic and lifestyle factors. It demonstrates a complete machine learning workflow from data preprocessing to model evaluation and prediction.

2. Dataset
File: Student_success_dataset_7000_rows.csv
Records: 7000

3. Features
- StudyHours
- Attendance
- PastScore
- Internet (Yes/No)
- SleepHours

4. Target
- Passed (Yes/No)

5. Tools & Technologies
Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn

6. Workflow
- Load and clean dataset
- Encode categorical variables
- Scale features
- Train-test split
- Train Logistic Regression model
- Evaluate using classification report and confusion matrix
- Predict result for user input
